📂 Ouvrir le dossier Drive

  1. Allez sur console.cloud.google.com → créez un projet (ou choisissez-en un existant)
  2. Menu APIs & Services → Bibliothèque → activez Google Drive API
  3. Menu APIs & Services → Identifiants → + Créer des identifiants → ID client OAuth 2.0
  4. Type d'application : Application Web
  5. Dans Origines JavaScript autorisées, ajoutez http://localhost et l'URL où vous ouvrez ce fichier (ex: file:// ou votre domaine)
  6. Copiez le Client ID (format xxx.apps.googleusercontent.com) et collez-le ci-dessus
  7. Dans Écran de consentement OAuth, ajoutez votre email comme utilisateur test si l'app est en mode test

⚠️ Le Client ID est stocké uniquement dans votre navigateur, aucune donnée ne transite par un serveur externe.
